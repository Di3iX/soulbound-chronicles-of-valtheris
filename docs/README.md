# Soulbound: Chronicles of Valtheris

A dark fantasy Action RPG that is being developed with a modular architecture and future MMORPG expansion in mind.

---

## Current Version

**v0.1.5**

---

## About

Soulbound: Chronicles of Valtheris is a dark fantasy RPG featuring exploration, combat, equipment, quests, merchants and character progression.

The project is designed from the beginning with a scalable architecture so it can eventually evolve into an MMORPG.

---

## Features

### Implemented

- Character progression
- Inventory system
- Equipment system
- Random item bonuses
- Merchant
- Quest system
- Multiple locations
- Enemy combat
- Boss battles
- Save / Load
- Character stats
- Talent window
- Persistent world progress
